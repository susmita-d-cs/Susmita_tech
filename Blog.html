Ontology Driven Edition Of Desision Trees

The aim of semantic web is to elaborate simple data on the world wide web to semantic layer thereby the knowledge processing by machine becomes simple and more shareable. Ontology is one the key technologies to understand and process semantic web. On the other hand, complex machine learning models have achieved notable success for prediction. The amalgamation of these two domains can lead to better interpretability, realistically achieve a better prediction result and understand the relationships among the attributes from ontology documents. The main goal of this thesis is to explore how information formalized in an ontology can be used together with machine learning datasets and algorithms for potentially improving interpretability. The specific objectives are:

• To develop a semi-automated pipeline that creates an ontology from a machine learning dataset for classification.

• To implement a functionality that suggests which attribute to use as a replacement in a decision tree node, based on the relations stated in the ontology.
